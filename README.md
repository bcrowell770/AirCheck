# AirCheck (Pebble Watch App)

A Pebble app that displays air quality and environmental data directly on your watch.

## Features

- Air Quality Index (AQI)
- Weather data integration
- Configurable settings via Pebble app

## Platform

- Pebble (Basalt / Aplite / Chalk)

## Installation

- Install via Pebble app
- Or sideload the `.pbw` file

## Status

Actively developed and evolving.
